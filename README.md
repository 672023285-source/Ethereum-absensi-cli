# Ethereum-absensi-cli

Smart Attendance System using Ethereum Smart Contract

## Deskripsi
Proyek ini merupakan sistem absensi berbasis Ethereum Smart Contract yang digunakan untuk mencatat data kehadiran secara transparan dan aman menggunakan teknologi blockchain.

## Teknologi yang Digunakan
- Solidity
- JavaScript
- Node.js
- Express.js
- Ethers.js
- HTML
- CSS

## Cara Menjalankan
1. Jalankan Anvil.
2. Install dependency dengan `npm install`.
3. Jalankan server menggunakan `node server.js`.
4. Buka aplikasi melalui browser.
